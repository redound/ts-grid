TSGrid 
=========

> Typescript Data Grid Library.

## Usage

```ts
import * as TSGrid from 'ts-grid'
```

## Contributing

```sh
# Installation
# Fork this repo (https://github.com/redound/ts-grid)
# Clone the fork (E.g. `https://github.com/<your_username>/ts-grid.git`)

cd ts-grid

```
